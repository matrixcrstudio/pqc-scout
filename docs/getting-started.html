Getting started

Install the package, run an assessment, read the roadmap. No API keys or accounts required for the default path.

Requirements

Python 3.10 or newer. The core install depends only on pydantic.

Install

# from a clone of the repository
pip install .

# optional: LLM-assisted extraction
pip install .[llm]

Assess a stack

Describe your stack in plain text (or point at a file). Deterministic extraction is the default — it reports only what it can literally identify, and says so.

pqc-scout assess --target "ExampleCo" \
  --stack "nginx TLS 1.3, RSA-2048 code signing, AES-128 backups"

# from a file, with a sector retention prior
pqc-scout assess -t "GenomicsLab" -s ./stack.txt --sector genomic

Scan live TLS endpoints

The TLS scanner performs a standard handshake against hosts you name. It is passive: no enumeration, no downgrade probing. Scan only infrastructure you operate or are authorized to assess.

pqc-scout scan example.org example.com

Read saved reports

pqc-scout list
pqc-scout score   -r <report-id>
pqc-scout roadmap -r <report-id>
pqc-scout status

Command reference

PQC-Scout commands
CommandPurpose
assessFull assessment from a stack description
scanLive passive TLS scan of one or more domains
scorePer-surface scoring for a saved report
roadmapPhased migration roadmap for a saved report
listRecent assessments
statusDatabase statistics and standards references

LLM-assisted extraction (--llm) requires the llm extra and an ANTHROPIC_API_KEY. It fails loud: a truncated or invalid model response aborts the assessment rather than silently falling back to the deterministic baseline. You always know which extractor produced a report.
